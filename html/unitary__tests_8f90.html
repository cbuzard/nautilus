program unitary_tests
 Program that test different functions implemented in nautilus. Especially routines that were added afterwards, and are not present in the original version, thus can't be compared between the two versions.

This program need a test simulation in the sub-folder 'tests'. Output results will be stored in the same folder.

This binary is conceived so that it is run by the python script unitary_tests.py.
